# PediatricCareCenter
Project Description: Pediatric Care Center System  The Pediatric Care Center project is a semester-long software engineering class project developed by Team 3 (Ivy Wang, Myung Cho, Kiera Feng, Baixi Jiao, and Rina Kawamura). The goal of the project was to design and implement a healthcare portal system focused on pediatric patient care.
